## Phase 2 — Design (Claude Design)

**Initial prompt sent to Claude Design:**

> "Create a Progressive Web App called 'Celestia' — a horoscope app with a dark, mystical aesthetic. Use a deep navy/dark purple color palette (#0D0D2B, #1A1A4E) with soft pink and lavender accents for highlights.
>
> The app should have 4 screens:
>
> 1. **Home Screen** — zodiac sign selector with illustrated sign cards (similar to a carousel), showing the sign name and date range
>
> 2. **Daily Horoscope Screen** — shows the selected sign's illustration prominently at the top, with tabs for Today / Tomorrow / Month, and circular progress indicators for Love, Emotions, Mentality and Career with percentages
>
> 3. **Compatibility Screen** — two zodiac avatars connected by a celestial orbit animation with floating stars in the background, and an 'Add Partner' button
>
> 4. **Settings Screen** — simple profile setup where user picks their sign and birth date, saved to Local Storage
>
> Navigation: bottom tab bar with icons for Horoscope, Compatibility, Profile
>
> Style: illustrated zodiac figures in a dark blue/purple duotone style, floating stars/particles in the background, rounded cards with glassmorphism effect, smooth transitions between screens"

**Reference images:** Two horoscope app screenshots provided showing dark UI with zodiac sign grids and energy indicators.

**Refinement prompts used in Claude Design:**
- "Add screen slide-in transitions on every navigation"
- "Bottom tab bar with glyph icons"
- "Custom status bar + home indicator"
- "Add traits to each zodiac sign in the settings screen"

**Output:** `Celestia.html`, `celestia-data.jsx`, `celestia-components.jsx`, `celestia-screens.jsx`

## Phase 3 — PWA Conversion (Claude Code)

**Prompt:**
> "Convert this Claude Design output into a full PWA. Add manifest.json, sw.js, PNG icons (192x192 and 512x512), update index.html with manifest link and service worker registration, create README.md, AGENTS.md, and .gitignore."

**Files generated by Claude Code:**
- `index.html` — added `<link rel="manifest">`, theme-color meta, apple-touch-icon, SW registration
- `manifest.json` — PWA manifest with name, icons, display: standalone, theme color
- `sw.js` — cache-first service worker for local assets
- `generate-icons.js` — pure Node.js PNG icon generator (no dependencies)
- `README.md` — project documentation
- `AGENTS.md` — this file
- `.gitignore` — excludes node_modules, .env, .DS_Store
